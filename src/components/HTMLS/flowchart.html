Procurement Flow: RFP → Quotation → PO → Invoice → QA → Payment

Setup

Company → Create company + addresses
Users → Register users linked to company with roles
Prereqs for all later steps

RFP

Create RFP with requirement type, delivery location/date
Add RFP products (quantities, descriptions)
Collaborate via queries and responses (threaded)
Approvers list records approvals

Quotation

Vendors submit quotations for the RFP
Line pricing per RFP product + taxes
Attach supporting documents; mark primary if preferred

PO

Create PO selecting the chosen quotation
PO references RFP, Quotation, Company, and User
Locks commercial terms

Invoice & QA

Vendor delivers; create Invoice(s) for the PO
Goods status: delivery + QA checks (QA leader)
Three-way match: PO ↔ GRN/Delivery ↔ Invoice

Payment

Approve invoice; process payment
Close loop and record audit trails
→ Governance throughout: audit trails per event and user
